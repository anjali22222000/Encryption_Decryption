# Encryption_Decryption
# 🔐 File Encryption and Decryption using AES (Java)

## 📌 Project Description (Point-wise)

1. The project provides secure **file encryption and decryption** using the **AES algorithm**.
2. **AES (Advanced Encryption Standard)** is a symmetric key algorithm, meaning the **same key is used for both encryption and decryption**.
3. The system uses a **16-bit (128-bit) secret key** for encrypting and decrypting files.
4. The user selects any file, and the program converts it into **encrypted form**, making the content unreadable.
5. During decryption, the user must enter the **same 16-bit key**, otherwise the file cannot be restored.
6. **Java is used as the backend**, implementing AES logic, file handling, and key processing.
7. The project uses Java’s **`javax.crypto`** package, including `Cipher`, `SecretKeySpec`, etc.
8. AES provides **strong security**, fast performance, and resistance to brute-force attacks.
9. The system ensures that **only users with the correct key** can decrypt and access the file.
10. This project demonstrates practical concepts of **cryptography**, **data security**, **symmetric key encryption**, and **secure file handling**.
